# 01. Docker Basic

## 컨테이너

1. 표준화된 소프트웨어 모임
2. 코드 패키지 및 코드를 실행하기 위한 종속성, 도구, 환경을 포함
3. 똑같은 컨테이너는 항상 정확하게 같은 환경에서 실행되게끔 함
4. 도커는 컨테이너를 구축하고 관리하기 위한 사실상 표준 도구
